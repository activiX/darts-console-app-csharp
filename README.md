# C# Darts Console Game
C# console darts game developed as a learning project. Demonstrates input validation, control flow, methods, structs, and basic game logic including multiplayer support and optional double-out rules.


## Features
- Supports only 501 game mode
- Single or multiplayer (up to 4 players)
- Optional double-out rule
- Input validation for dart throws
- Checkout suggestions
- Turn and round management

## Purpose
This project was created as a learning exercise to practice C# fundamentals such as:
- control flow
- input validation
- methods and structs
- basic game logic

## Requirements
- .NET 9.0
  
## How to run
Run the application as a standard C# console app.
